This folder contains all visual assets for the Seasonal Race viewer.

See docs/ASSET_CREATION_GUIDE.txt for complete instructions on
what to create, what each asset should look like, and which free
tools (Microsoft Designer, Canva, GIMP, Piskel) to use.

Folder structure:
  backgrounds/   - 4000x800 wide panoramic race scenes (PNG/WebP)
  textures/      - Tileable surfaces + displacement maps (PNG)
  particles/     - Small particle textures 8-24px (PNG, transparent bg)
  racers/        - Spritesheet PNG + atlas JSON per racer type
  props/         - Obstacles and decorations (PNG, transparent bg)

Until real assets are placed here, the viewer shows colored
placeholder rectangles automatically — so development can continue
in parallel with asset creation.
